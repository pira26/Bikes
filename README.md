# Fullstack Bike Store RESTful Project 

Using : 

- React Js 
- Node Js / Express
- MongoDb / Mongoose

- Material UI

- Webpack

- Heroku

After the git clone : 

```
$ npm i

``` 

Open 2 terminals and run the below code with your local machine... 

```
$ npm start
$ npm run bundle

``` 

... or go to: 

```
https://evening-caverns-85699.herokuapp.com/

```
